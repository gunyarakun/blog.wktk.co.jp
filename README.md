# blog of wktk inc.

This repository contains sources of blog of wktk inc., japanese company.

- http://blog.wktk.co.jp/

## How to install

  > git clone https://github.com/gunyarakun/blog.wktk.co.jp.git
  > cd blog.wktk.co.jp
  > gem install bundler -v 2.4.22
  > bundle install

## How to build

  > rake preview

## How to deploy

```
> rake publish
```
